This is a repository for supplemental materials for the research article "Unpacking teacher collaboration: validation of measurement models using exploratory structural equation modeling" by Waldejer et al. (2026).
In the zip-folder you will find another README will further details, R-scripts, data and corresponding codebook.
To avoid issues with filepaths/working directories, please download the zip folder and avoid changing the file structure in the folders.

Note that R version 4.1.0 or later is required to run the scripts without having to make changes.
